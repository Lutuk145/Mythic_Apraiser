# Mythic_Apraiser
## Mod for wynncraft that adds weighted ids to mythic weapons and armor using this (outdated) list for not https://forums.wynncraft.com/threads/mythic-stat-order-guide.303507/
## If you are interested in helping to update some weapons message me on discord @lutuk, if you have any idea how to improve this mod also please message me and Ill be interested in hearing what you have to say
